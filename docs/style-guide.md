# Design System & Style Guide

This document provides an overview of the design standards and reusable components.

## Colors

- Primary: #0055FF
- Secondary: #FF5500
- Background: #FFFFFF
- Text: #000000

## Typography

- Heading: Roboto Bold, 24-32px
- Body: Roboto Regular, 14-18px
- Line height: 1.5

## Icons & Assets

- All icons should be SVG format, 24x24px.
- Images should be optimized for web and retina displays.

## Components

- Buttons, cards, modals, form inputs.
- Components should have consistent padding, margins, and shadows.

## File Naming

- Templates: `component-name-v1.fig`
- Assets: `icon-name-v1.svg`, `illustration-name-v1.png`
